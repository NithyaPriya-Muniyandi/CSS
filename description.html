HTML Terms

HTML
HTML stands for HyperText Markup Language and is used to create the structure of a web page.
HTML element
An HTML element is a fundamental building block of a web page, represented by tags, and it can include content and other elements.
HTML tag
An HTML tag is used to define an HTML element. It consists of an opening tag and a closing tag, enclosed in angle brackets.
Angle bracket
An angle bracket (< and >) is a punctuation mark used in HTML to enclose tags and elements.
HTML attribute
An HTML attribute provides additional information about an HTML element and is defined within the opening tag of the element.
HTML comment
An HTML comment is a way to add notes or comments to HTML code that are not displayed in the web page's output.
HTML object element
The HTML object element is used to embed external resources like multimedia or interactive content in a web page.
HTML special character
HTML special characters, also known as entities, are used to display reserved characters or symbols in HTML, like < for < and > for >.
